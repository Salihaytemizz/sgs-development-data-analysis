# SGS - Smart Growth Solutions

🚀 **Ultra-basit veri analizi - SQL kadar güçlü!**

Tek komutla Excel ve veritabanı dosyalarınızı analiz edin. 20+ detaylı bulgu, trend analizi, performans skorları.

## ✨ SGS Power Özellikleri

- **Tek komut analizi**: `sgs.analyze()` 
- **SQL kadar güçlü**: 20+ detaylı bulgu
- **Akıllı tespit**: Otomatik veri türü tanıma
- **Trend analizi**: Yükselen/düşen ürünler
- **Şube karşılaştırması**: Çoklu lokasyon analizi
- **Fırsat tespiti**: Popüler ama eksik ürünler

## ✨ Features

- **One-line analysis**: `sgs.analyze('restaurant_data.xlsx')`
- **Restaurant-focused insights**: Menu optimization, pricing strategy, performance analysis
- **Automated reporting**: Beautiful HTML reports with actionable recommendations
- **Privacy-first**: All processing happens locally on your machine
- **Zero setup**: No complex configurations or API keys required

## 🚀 Quick Start

```python
import sgs_prototype as sgs

# Analyze your restaurant data
sgs.analyze('your_restaurant_data.xlsx')

# Get instant insights and recommendations
# Report saved as: sgs_restaurant_report.html
```
